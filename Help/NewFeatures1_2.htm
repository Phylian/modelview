New Features in Version 1.2

  • Support for PHAST has been added.
  • Support for _SOS files produce by UCODE-2005 has been added.
  • Support for SUTRA 2.1 has been added.
  • Several bugs in Model Viewer 1.1 have been fixed.

Note: MV files saved by previous versions of Model Viewer can be opened in version 1.2.
