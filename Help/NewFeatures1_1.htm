New Features in Version 1.1

  • A line drawing contained in an ESRI Shapefile or a DXF file may be displayed on the top surface of the model grid or on a flat plane above the model grid.
  • Display of particle locations in MODFLOW-2000 GWT and MOC3D
  • Support the final release of SUTRA version 2D3D.1. The beta version of this model is no longer supported.
  • A number of bugs in Model Viewer 1.1-1.3 have been fixed.

Note: MV files saved by Model Viewer 1.0 can be opened in version 1.1.

New Features in Version 1.2

  • Support for _SOS files produce by UCODE-2005 has been added.
  • Support for SUTRA 2.1 has been added.

New Features in Version 1.3

  • Model Viewer has been altered to allow the display of heads and drawdowns saved as binary files with double-precision real numbers.
  • Added drains model feature for phast.
  • Added double-click functionality to ModelFeaturesDlg list boxes.

New Features in Version 1.4

  • Added support for SUTRA 2.2
  • Added the ability to make an animation that does not advance with time.
  • Added support for MODPATH version 5 pathline files.
  • Added capability of log-transforming color values for pathlines.
  • Added capability of reading MT3DMS double-precision binary files.
  • Added the abililty to display vectors for double-precision MODFLOW models.

New Features in Version 1.5

  • Fixed bug that caused Model Viewer to quit improperly when models with numerous parameter instances are used.
  • Modified color bar to allow it's data to come from either the active data set or the pathlines.
  • Made changes to allow Model Viewer to read the input files for the LAK and SFR packages that use the revised input format in MODFLOW-2005 version 1.9.

New Features in Version 1.6

  • Model Viewer can now display pathlines generated by MODPATH version 6.
  • With MODFLOW-2000 and MODFLOW-2005 models, Model Viewer can now display data from external files in addition to files generated by MODFLOW.
