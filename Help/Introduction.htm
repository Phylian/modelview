Introduction

Model Viewer is an application that displays the results of three-dimensional ground-water models.

Model Viewer version 1.7 supports the following models:

  • MODFLOW-2005
  • MODFLOW-2000
  • MODFLOW-2000 with Ground-Water Transport (GWT) Process
  • MODFLOW-96
  • MOC3D (Version 3.5)
  • MODPATH
  • MT3DMS
  • PHAST
  • SUTRA (Versions 2.2, 2.1, and 2.0)
  • UCODE-2005

Model Viewer is designed to directly read input and output files of the above listed models, thus minimizing the need for additional postprocessing.
